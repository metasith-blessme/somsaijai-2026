KEY POINTS TO REMEMBER
================================================================================

✓ Image numbers (1-28) are NOT the same as calendar dates
  → Always read the date from the actual image content

✓ All amounts are in Thai Baht (฿)
  → No currency conversion needed

✓ Each product has TWO amounts: Cash and Scan
  → "Scan" typically means digital payment (card transfer, etc.)

✓ Some products (Mango, Coconut) may not appear daily
  → Only enter if present in that day's image

✓ Total Cups = Orange cups + Watermelon cups + Mango cups (if any)
  → Must equal the sum of all cup product sales counts

✓ Expenses are tracked
  → Primarily ice costs
  → Other operating expenses if listed
